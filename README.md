Stop Watch – Modern Card Style

A simple Stop Watch built with HTML, CSS, and JavaScript, featuring:

Card-style layout with shadowed, rounded container

Start, Stop, Reset buttons with hover and active animations

Keyboard accessibility: tab through buttons + shortcuts (S = Start, T = Stop, R = Reset)

Screenreader-friendly: live timer updates with aria-live="polite"

Responsive design for mobile and desktop

Minimal and clean styling for modern look

🔹 Features

Incremental timer with seconds and minutes

Barrierefrei / Accessible (ARIA attributes, focus outlines)

Buttons have visual feedback and spacing (no clutter)

Works without mouse, fully keyboard-operable

Easy to integrate into any web page

🔹 Usage

Clone or download the repository:

git clone https://github.com/yourusername/stopwatch-card.git

Open index.html in a browser

Use the buttons or keyboard shortcuts:

S = Start

T = Stop

R = Reset
